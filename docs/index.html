记事本介绍#

记事本是一个本地 HTML 文档集合，用于组织和浏览日常记录下来的经历、思想、感悟等。它实现了：

  • 桌面环境（电脑）和移动环境（手机）自动适应。
  • HTML 和 Markdown 混合编辑。
  • 文档分目录管理。
  • 多级菜单，并且菜单栏基于当前页面自动展开和高亮。

使用配置#

  1. 将记事本代码，通常是notebook文件夹，放到您经常访问和编辑的位置。

    注意：如果记事本代码不在notebook目录，记得修改jac/config.js文件中的rootPath配置项。

  2. 使用浏览器访问记事本目录中的index.html文件，浏览记事本首页。您也可以把该文件添加为书签或收藏，以便在以后随时访问。
  3. 在记事本目录（及其子目录）中任意添加或编辑*.html文件。您现在已经创建了属于您自己的记事本。
  4. 对于新建的文档，修改jac/config.js文件中的menu配置项以便添加到左侧菜单栏。
