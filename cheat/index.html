简要介绍#

因为各类语法、命令的内容涵盖广泛，重点内容淹没于其中，导致实际使用过程中，也许 80% 的知识都用不上，而真正需要用到的知识却找不到，所以制作此速查手册。

速查手册只记录最常用的语法和命令，并只做简短的介绍，便于用户快速查阅。

Markdown 介绍#

Markdown 是一种轻量级的标记语言，通过简单的标记语法，就可以实现漂亮的排版。Markdown 具有一系列衍生版本，用于扩展其功能（如表格、脚注、内嵌HTML等等）。记事本默认支持 GitHub Flavored Markdown 格式，其完整的语法可以在GFM 官方说明找到。

需要注意的是，记事本虽然使用标准的 Markdown 语法，但解析方式依照 Spectre.css 的样式重新进行了定义，详见记事本定制。

Git 介绍#

Git 是一个开源的分布式版本控制系统，可以敏捷、高效地处理基于文本的大小项目，同时也可以作为内容管理系统，工作管理系统等。Git 官网上提供了关于 Git 的完整手册和文档（含中文文档），还有各种类型的 Git 客户端供下载使用。

通常我们不需要建立自己的代码服务器，而是将项目存放到专业托管平台。在这些代码托管平台中，Github可以作为您托管项目的首要选择，它对开源项目免费，但建立私有项目则需要收费。如需免费建立私有项目，可选择GitLab或Bitbucket。

注意：对这三个项目本身而言，只有 GitLab 是开源的，但它的官网发生过丢失数据的事故。
